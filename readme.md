# Mega Mall Online Store UI/UX Redesign

This project is a UI/UX redesign concept for an ecommerce store focused on creating a cleaner, more premium, and more mobile-friendly shopping experience.

## Deployed Link

https://owethujezilereachout.netlify.app

## Project Overview

The redesign improves the browsing experience by refining navigation, product presentation, mobile interactions, and visual hierarchy while preserving the brand identity and core ecommerce flow.

## Key Features

- Modern responsive navbar and mobile navigation drawer
- Clean product cards with stronger visual hierarchy
- Wishlist and cart experience improvements
- Premium e-commerce-inspired layout and spacing
- Accessible and touch-friendly interface patterns

## Technologies Used

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
- Lucide React
- ESLint

## Project Structure

- src/components - reusable UI and page components
- src/pages - route-level pages such as Home, Products, Cart, and Wishlist
- src/context - cart and wishlist state management
- src/data - product and category data
- src/assets - images, icons, and other static assets

## Getting Started

1. Clone the repository
2. Install dependencies with npm install
3. Start the development server with npm run dev

## Notes

This project is a portfolio-style redesign and is intended to demonstrate modern UI/UX improvements for an ecommerce experience.
